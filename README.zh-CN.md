# 软件工程 Skills

面向软件工程工作的 Codex 技能合集，欢迎社区共建。

## 已包含技能

- `rust-best-practices`
- `go-best-practices`
- `se-collab-standards`

## 目录结构

```
skills/
  rust-best-practices/
  go-best-practices/
  se-collab-standards/
```

## 快速开始

1. 克隆本仓库。
2. 将需要的 skill 目录复制到你的 Codex skills 目录。

示例路径：

- Windows：`C:\Users\<you>\.codex\skills`
- 自定义：`$CODEX_HOME\skills`
